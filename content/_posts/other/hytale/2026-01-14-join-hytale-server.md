{: .warning }
> Hytale server hosting requires the official Hytale Game to be purchased.

## Starting Your Server

Before you can join, make sure your server is running:

1. Log in to the [Dashboard](https://client.falixnodes.net/) and select your Hytale server from the server list.

2. Make sure that your server is running and press the **Connect**{: .blue } button (located next to the start/stop buttons).

3. Copy the "Domain Address".

4. Open the Hytale game and go to the "servers" menu.

5. Click the **Add server** button.

6. Enter the Domain Address in the top (connection address) field.

7. Give the server a name in the second field and click the **Add server** button.

8. Double click the server to join.
