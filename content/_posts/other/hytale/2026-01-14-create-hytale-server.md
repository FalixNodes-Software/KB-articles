## Introduction

Welcome to the world of Hytale! This guide will walk you through creating your very own Hytale server on Falix, allowing you to build, explore, and adventure with friends in this blocky universe.

{: .warning }
> Hytale server hosting requires the official Hytale Game to be purchased.


## Creating Your Hytale Server


1. Log in to the [Falix Dashboard](https://client.falixnodes.net/).

2. Click on [Create Server](https://client.falixnodes.net/create) in the navigation menu.

3. Choose the **Premium** plan and click **Continue**.

4. Enter a name and domain for your server, and click **Continue**.

5. Select "Hytale" and click **Continue**.

6. Enter the amount of resources you want to allocate to the server.

7. Scroll down, select the location closest to you and click **Continue**.

8. Review if everything is correct and press **Create My Server**.

## Starting Your Server

1. Return to the [Dashboard](https://client.falixnodes.net/).

2. Select your newly created Hytale server in the server list.

3. Wait for the server to be installed completely and press **Start**.

4. Wait until it shows a "Hytale Account Login" popup and then press **Log in**.

5. Click **APPROVE** and sign in using your preferred method if needed.

6. Wait for the server to start up and join the server like explained [here](/other/hytale/general/join-server).

7. Open the console again and click the **Link account** option (if needed).

8. Wait for it to show a new login prompt and log back in to your Hytale account.

9. Run the `/auth persistence Encrypted` command in the console to store the account credentials.

10. Click the **retry** option in the game or join the server again.
